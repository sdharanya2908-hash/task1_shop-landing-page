# Sweet Treats Bakery Landing Page

This is a **Single Page Responsive (SPR)** website for a bakery shop named **Sweet Treats Bakery**.  
It is built using **HTML** and **CSS** only.  

## Features

- **Hero Section**: Full-width hero image with overlay text and "Order Now" button.  
- **Products Section**: 4 products with background images and prices in Indian â‚¹.  
- **About Section**: Fixed background image with overlay and text.  
- **Gallery Section**: Display 4 product images (optional).  
- **Contact Section**: Contact details and address.  
- **Responsive Design**: Works on desktop, tablet, and mobile devices.  

## Technologies Used

- HTML5
- CSS3

## How to Use

1. Clone or download the repository.
2. Open `index.html` in your browser to view the website.
